Twitch 스트리머 및 시청자를 위한 대량의 기능을 지원합니다!
(이 모드는 메가크릿 및 Twitch와 무관한 서드 파티 모드입니다)

[h1]더 이상의 "저 유물 뭔가요?"나 "저 포션 뭐에요?" 같은건 없습니다 :P[/h1]

대부분의 명령어 및 메세지들은 직접 수정 가능합니다 :)
Windows 기준 %LOCALAPPDATA%/ModTheSpire/TwitchUtils 내에 있는 json 파일을 수정하시면 됩니다.
상세한 수정 방법은 해당 파일 내에 적어두었습니다!


해당 모드를 사용하시려면 기본적으로 슬레이 더 스파이어에서 지원하는 트위치 모드를 사용하셔야 합니다
관련된 [url=https://tgd.kr/18696623]트게더 글[/url] 을 참고하세요.
또한 트위치 모드에서 기본적으로 제공하던 카드 및 보스 유물 투표 기능을 비활성화합니다.
해당 기능은 게임 내의 Mods 화면에서 켜고 끌 수 있습니다.


[h1]지원하는 기능[/h1]
[list]
[*] [b]모든 현재 유물 확인[/b] 
!현재유물, !cr

[*] [b]특정 현재 유물 확인[/b] 
!현재유물 <숫자>, !cr <숫자> (!cr 1 -> 가장 왼쪽 유물, !cr -1 -> 가장 오른쪽 유물)

[*] [b]모든 현재 포션 확인 [/b]
!현재포션, !cp

[*] [b]특정 현재 포션 확인 [/b]
!현재포션 <숫자>, !cp <숫자> (!cp 1 -> 가장 왼쪽 포션)

[*] [b]모든 상점 유물 + 특정 상점 유물 확인[/b]
!상점유물, !sr  (기타 상동)

[*] [b]모든 상점 포션 + 특정 상점 포션 확인 [/b]
!상점포션, !sp (기타 상동)

[*] [b]현재 덱리스트 확인[/b]
 !덱리, !dl

[*] [b]모든 현재 사용중인 모드 확인 [/b]
!현재모드, !cm
!cm twichUtils, !cm tWiTch Utils, !cm twi -> TwitchUtils 하나에 대한 설명을 보여줍니다

[*] [b]특정 현재 사용중인 모드 확인 [/b]
!현재모드 <모드이름>, !cm <모드이름>

[*] [b]기본 트위치 모드에 적용되는 카드 및 보스유물 투표 기능 제거[/b]
.json 설정 파일 및 게임 내의 Mods 탭에서 비활성화를 켜고 끄실 수 있습니다
[/list]


[h1]커스터마이징 지원[/h1]
[list]
[*] 현재유물, 현재포션, 상점유물, 상점포션, 덱리스트, 현재모드 명령어에 대한 명령어(접두어), 메세지(출력형식), 온/오프, 쿨타임
[*] 현재 사용중인 모드들에 대한 커스텀 설명 기능
[*] %LOCALAPPDATA%/ModTheSpire/TwitchUtils
[*] 이 모드를 켜시고 (!현재모드 명령어에서 모드 설명을 임의로 추가하시려면 해당 모드까지 같이) 한 번 실행하시면 위 경로에 .json 설정 파일들이 생성됩니다
[*] 상세한 설정 파일 수정 방법은 해당 설정 파일들 내에 설명되어 있습니다
[*] 현재 영어, 한국어에 대한 기본 설정값을 제공합니다(최초 실행 시 슬더스 언어 설정에 따라서)
[/list]
